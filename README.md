# Coca-Cola Stock Price Prediction (ML Project)

This project analyzes **Coca-Cola (KO)** historical stock data and builds a **machine learning model** to predict future stock prices.  
It includes data collection, exploratory data analysis (EDA), feature engineering, model training, evaluation, and visualization.
